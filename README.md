# PathoLab
Languages used: Java, Servlets, JSP, SQL. 
Description: PathoLab is an online pathology platform designed to enhance remote healthcare 
services by allowing doctors to efficiently manage patient consultations and prescriptions. The 
platform allows doctors to access detailed patient records, including medical histories and lab 
reports, enabling accurate diagnoses and treatment plans. The system ensures secure data 
handling, real-time updates, and an intuitive interface for seamless interaction between doctors 
and patients. Additionally, it streamlines the prescription process, reducing the need for in
person visits and improving accessibility to medical care.
